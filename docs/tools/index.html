LittleCub Forms

Light-Weight, High-Performance & Easy Customizable

  • Home >
  • Tools

LittleCub is a light-weight, high-performance, easy customizable forms engine for Node.js and browser. It comes with support for various responsive and mobile UI frameworks and can be easily customized or extended.

LittleCub Forms is based on JSON schema V4 and can be used to present JSON data as entry or edit forms
